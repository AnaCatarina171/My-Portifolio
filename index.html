Who am I?

      I, Ana Catarina Ribeiro Silva, am a young Brazilian who moved to Canada looking for a better future in my professional life.
      During high school, I started questioning myself about my career path. On a random afternoon, a YouTube video on learning JavaScript appeared on my explore page, which caught my attention. I had never studied programming before, but after watching that video, I was amazed by how much we can create with coding, and it was not as complicated as I thought it would be. After watching a few videos from this course, I fell in love with programming and decided to make it my career.
      Moving to another country was always my dream, so, on August 2020, I applied to study at the New Brunswick Community College, in Canada.
      On April 2021, I started taking Information Technology: Programmer-Analyst at NBCC. During my studies, we focused intensively on Object-Oriented Programming, especially using C# and Java, with which I built many applications with .NET and Spring Boot frameworks. Through Web Design courses, I constructed responsive web applications that included real-world functionality, so I have excellent knowledge of HTML5, CSS3, and JavaScript.
      At the beginning of 2021, I completed an Oracle Database Course, which included SQL, PLSQL, and Data Modeling, giving me excellent database programming skills.
      As part of my Work Practicum, from October 2022 until December 2022, I built an entire Spring Boot web application for the cafeteria located on NBCC Moncton Campus, using JavaScript and Thymeleaf as the front end, and Oracle 11g as the database. This project can be accessed through my GitHub account.
      So, now, I am excited to keep thriving in my developer career and looking for a Junior Developer position.
